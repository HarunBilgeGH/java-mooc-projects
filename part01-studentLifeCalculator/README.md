# Student Life Calculator

A simple Java console project developed after **Part 01** of the Helsinki MOOC Java Programming course.

This is a beginner-level practice project. The goal is to reinforce basic Java concepts, not to build a complex application.

## Features

* Reads student information from user input
* Checks age status
* Calculates grade and pass/fail result
* Calculates monthly budget
* Checks scholarship eligibility
* Checks basic gift tax status
