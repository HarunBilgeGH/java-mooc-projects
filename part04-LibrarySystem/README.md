# Library Management Console App

Library Management Console App is a beginner-level Java console project developed to practice Helsinki MOOC Java Programming Part 4 concepts.

The project reads book data from a text file, stores books as objects in an `ArrayList`, and allows the user to manage books through console commands.

## Features

- Read book data from a file
- Store books as objects
- List all books
- Search books by title
- Search books by author
- List long books
- Borrow books
- Return books
- List available books
