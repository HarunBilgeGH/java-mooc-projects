# Movie Watchlist Analyzer

Movie Watchlist Analyzer is a simple Java console application developed as a practice project for Helsinki MOOC Java Programming Part 3.

The project allows users to add movies, list saved movies, search movies by title, find the highest-rated movie, calculate the average rating, and display movie genres.

## Features

- Add movie records
- List all movies
- Search movies by title
- Find the best-rated movie
- Calculate average movie rating
- Print movie genres


## Movie Input Format

Movies must be entered in the following format:

```text
movieName,year,rating,genre
