# Course Enrollment System

This is a practice project inspired by Helsinki MOOC Java Programming Part 5. The goal is to strengthen object-oriented programming fundamentals, especially objects, references, object fields, method parameters, `ArrayList`, `equals`, and methods that return objects.

## Project Idea

The project models a small course enrollment system.

A student can enroll in a course if they have enough credits. Successful enrollments are stored by a study office. Each enrollment keeps track of the student, the course, and the date of enrollment.

## Features

Student and course management
Credit-based enrollment eligibility
Course enrollment tracking
Enrollment history by student
Latest enrollment lookup
Rejected enrollment counting
Simple custom date handling
Object-oriented class relationships
